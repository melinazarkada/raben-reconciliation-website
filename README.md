# Raben Reconciliation Website

Static HTML/CSS/JavaScript prototype for the Raben Logistics reconciliation workflow.

## Pages

- `index.html` - login and quick access page
- `reconciliation.html` - reconciliation dashboard
- `operations.html` - shipment entry screen
- `mydata.html` - MyData approval screen
- `integrations.html` - integrations status and logs

## Run Locally

Open `index.html` in a browser. No build step is required.
